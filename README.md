# Hanami
<h1>About</h1>
<p>花見「はなみ」(Hanami) is the Japanese tradition of watching cherry blossoms. The cherry bloom usually lasts from early March to late May.</p>
<p>This theme has been designed for @<a href="https://codingcabin.tumblr.com/" title="codingcabin" target="_blank">codingcabin</a>&rsquo;s <a href="https://codingcabin.tumblr.com/post/183138356617/gridmania-challenge-coding-awards" title="gridmania challenge by codingcabin" target="_blank">gridmania challenge</a>, which is part of their on-going <a href="https://codingcabin.tumblr.com/awards#_=_" title="coding awards by codingcabin" target="_blank">coding awards</a> series.</p>
<h1>Features</h1>
<h2>Layout</h2>
<ul>
<li>Contained layout.</li>
<li>Sidebar.</li>
<li>One or two columns.</li>
<li>Three post placements (two columns only!).</li>
<li>Responsive layout.</li>
</ul>
<h2>Sidebar</h2>
<ul>
<li>Background image.</li>
<li>Title, description,&nbsp;and icon.</li>
<li>Navigation panel.</li>
<li>Five optional custom links.</li>
<li>Search bar.</li>
<li>Three options for ordered lists.</li>
<li>Customizable unordered lists.</li>
</ul>
<h2>Design</h2>
<ul>
<li>Two main font options.</li>
<li>Two options for quote/title fonts.</li>
<li>Three font-size options.</li>
<li>21 color options.</li>
</ul>
<h2>Extras</h2>
<ul>
<li>Custom audio background.</li>
<li>Custom mobile toolbar.</li>
<li>Custom scrollbar.</li>
<li>Custom tooltips.</li>
<li>Custom text selection.</li>
<li>Custom tumblr controls.</li>
</ul>
<h2>Accessibility</h2>
<ul>
<li>Tooltips on links, forms, and buttons.</li>
<li>Icons will not be read out loud by screen readers.</li>
<li>Deactivate italics at choice.</li>
<li><strong>Grayscale</strong> and <strong>Grayscale Hover</strong> allow you to keep the saturation of images low.</li>
<li>Change text alignments from justified or center-aligned to left-aligned.</li>
</ul>

<h2>Grayscale</h2>
<p><img src="https://66.media.tumblr.com/781d6f60ade35534583d48a143e7b050/tumblr_inline_pqmc1uwfi71v0e5mg_500.png" /></p>
<p>With the text fields <strong>Grayscale</strong> and <strong>Grayscale Hover</strong> it is possible to set the grayscale filter value of images in photo posts. <strong>Grayscale Hover</strong> affects the image when you hover over it, e.g. you could set the value to "10" if you don't want an image to have full saturation on hover.</p>
<p><em><strong>Important</strong></em>: You&nbsp; have to enter a value between 0 and 100 in those fields! Units do not have to be written here!</p>
<h2>Links</h2>
<p><img alt="image showing the text fields for a link's url and title" src="https://66.media.tumblr.com/98ecc2756bb89f67d010e3efa00c8fc7/tumblr_pqn0s89Icw1wsskx3o1_540.png" /></p>
<p>To set the custom links, you will need to enable them with the toggles first (enabled = blue). Then, further down the customization panel, there are text fields titled <strong>Custom URL#</strong> and <strong>Custom Link#</strong>.</p>
<p>Under <strong>Custom URL#</strong>, you have to add the URL of the link; under <strong>Custom Link#</strong> the name of the link.</p>
<h2>Lists</h2>
<h3>Ordered Lists</h3>
<p>To customize the ordered lists, click on the selector titled "Ordered List" and select a list style type out of the three given options.</p>
<h3>Unordered Lists</h3>
<p><img alt="image of the text field for unordered lists" src="https://66.media.tumblr.com/acd359ef0e35350d9621d623540f4c74/tumblr_pqn0s89Icw1wsskx3o2_540.png" /></p>
<p>For the unordered lists, there is a text field into which you have to enter the unicode of the icon you would like to use for the list. <a href="https://honeybee.suiomi.com/" title="suiomi | icon font honeybee" target="_blank">You can find the icons and their unicodes here</a>.</p>
<h2>Selectors</h2>
<p><img alt="image of all selectors" src="https://66.media.tumblr.com/8cd2ebdd97eb9ad0b6d1360dd29d2c42/tumblr_pqn0s89Icw1wsskx3o4_540.png" /></p>
<p>The selectors that you can see in the customization panel have the following purposes:</p>
<ul>
<li><strong>Font</strong>: Select between two font options (Noto Sans SC, Sawarabi Gothic).</li>
<li><strong>Font Size</strong>: Select between three font-sizes (16px, 14px, 12px).</li>
<li><strong>Italics</strong>: Activate or deactivate italics.</li>
<li><strong>Ordered List</strong>: Select between three list style types (decimal, hiragana, katakana).</li>
<li><strong>Post Placement</strong>: [<em>Only active when you toggle "Two Columns" on!</em>] Select between three post placements &ndash; top aligned, center aligned or bottom aligned.</li>
<li><strong>Quote Font</strong>: Select between two fonts for titles and quotes (Noto Serif SC, Sawarabi Mincho).</li>
<li><strong>Text Align</strong>: Select between two text aligns (justified, left-aligned).</li>
<li><strong>Title Align</strong>: Select between two alignments for quotes and titles (center-aligned, left-aligned).</li>
</ul>
<h2>Toggles</h2>
<p><img alt="image of all toggles" src="https://66.media.tumblr.com/ba4d561cfda0c13ad9363a28d06bf589/tumblr_pqn0s89Icw1wsskx3o3_540.png" /></p>
<p>Under the customization panel, there are multiple toggles that can be turned on (blue) and off (gray):</p>
<ul>
<li><strong>Show Captions</strong>: show or hide captions on index.</li>
<li><strong>Show Link[1-5]</strong>: show or hide custom links.</li>
<li><strong>Show Tags</strong>: show or hide tags on index.</li>
<li><strong>Show two columns</strong>: show posts in a one column or two columns layout.</li>
</ul>
